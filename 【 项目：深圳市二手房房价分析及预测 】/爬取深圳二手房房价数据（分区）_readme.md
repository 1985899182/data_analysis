## **背景描述**
爬虫爬取深圳二手房房价等相关数据，分10各区，共18514条数据

## **数据说明**
|    列名   |  列名含义  |                                    备注                                    |
|:---------:|:----------:|:--------------------------------------------------------------------------:|
| district  | 房屋所在区 | 深圳十个区：龙岗、龙华、宝安、南山、福田、罗湖、坪山、光明、盐田、大鹏新区 |
| roomnum   | 房间数     | 取值1~9                                                                    |
| hall      | 厅数       | 取值0~5                                                                    |
| AREA      | 面积       | 单位：㎡                                                                   |
| C_floor   | 楼层       | 分三档：middle、high、low                                                  |
| school    | 是否学区房 | 0（不是学区房），1（学区房）                                               |
| subway    | 地铁       | 0（不靠近地铁），1（靠近地铁）                                             |
| per_price | 平米单价   | 单位：万元/㎡                                                              |

## **数据来源**
搜房网 深圳 各区 二手房数据

## **问题描述**
通过对数据的清洗及处理，筛选对房价有显著影响的特征变量，对假设结论进行检验，确定特征变量，建立房价预测模型，并对假设情景进行模拟
