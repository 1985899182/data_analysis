## **背景描述**
* 这是一套来自某银行真实数据集构建贷款违约预测模型
* 源自kaggle项目🔗：https://www.kaggle.com/pranati25/predict-loan-defaulters
* 数据时间相对较老，共有8个csv文件：
				accounts(账户表)
				card(信用卡)
				client(客户信息表)
				disp(权限分配表)
				district(人口地区统计表)
				loans(贷款表)
				order(支付命令表)
				trans(交易表)
* 数据丰富，用作练手项目还是不错的。

## **数据说明**
* 一个客户可以拥有更多的账户，更多的客户可以操作单个账户;客户和账户在“disp(权限分配表)”中是相关的。
* “loans”表和“card”表描述了银行向客户提供的一些服务。
* 可以向一个帐户发出多张信用卡。
* 一个账户最多只能发放一笔贷款。
* “district”表提供了一些有关地区的公开信息(如失业率);可以由此推断出关于客户端的其他信息。


## **问题描述**
* 该数据集可以用于构建 信用卡客户画像 及 贷款违约模型
